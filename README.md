# LLM Tool Agent（

一个基于大语言模型（LLM）的 **Tool Agent 示例项目**，  
从零实现了 Agent 核心循环、工具调用机制与工程化结构，
用于学习和理解 LLM Agent 的基本工作原理。

---

## 项目简介

本项目实现了一个最小但完整的 LLM Agent 系统。  
Agent 能够根据用户问题进行推理，并在合适的情况下调用外部工具，
最终整合工具返回结果生成答案。

项目不依赖复杂框架（如 LangChain），而是采用**清晰、可读的代码结构**
从底层展示 Agent 的核心机制，适合作为：

- LLM / Agent 学习项目
- 工程化智能体实现示例
- 大模型方向实习的展示项目

---

## 核心特性

- 从零实现 **Agent Loop（思考 → 行动 → 观察 → 决策）**
- 基于 JSON 协议的 **工具调用机制**
- 支持多工具扩展（计算、时间等）
- 工具调用异常兜底，避免 Agent 崩溃
- 限制最大推理步数，防止无限循环
- 清晰的工程分层，便于理解和扩展
- 不暴露中间推理过程，符合实际工程实践

## 工具说明

当前内置工具包括：
计算工具（calc）：用于处理简单数学表达式
时间工具（time）：用于获取当前时间
工具采用统一的 JSON 返回结构，方便后续扩展更多能力。\

## 设计说明

Agent 在内部进行推理，但不会向用户暴露中间思考过程
工具调用结果以结构化数据形式返回，增强稳定性
项目强调可读性与可理解性，而非模型性能或复杂功能
